Data Orchestration Perspective

Icon:

Keyboard shortcut: CTRL-Shift-D

Description

The Data Orchestration perspective is where you’ll spend most of your time in Hop Gui. This is where pipelines and workflows are built, tested and run.

Usage

Pipelines and workflow files can be opened or created. At that point they appear in new tabs in this perspective.

For more information on how to get started with pipelines, see the Pipelines getting started guide.

For more information on how to get started with workflow, see the Workflows getting started guide.
